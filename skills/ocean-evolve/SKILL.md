## 📊 安全等级说明

| 等级 | 名称 | 说明 |
|------|------|------|
| **L0** | 无需告知 | 可立即执行，结束后正常记录即可 |
| **L1** | 执行后告知 | 可立即执行，执行后主动告知用户本次操作及结果 |
| **L2** | 必须请示 | 必须获得用户明确授权后才能执行 |
| **L3** | 禁止/临时授权 | 除非用户明确说"临时授权 L3"，否则绝对不可执行 |

**通用原则：若某操作未在下方表中列出，默认按 L2（必须请示）处理，然后询问用户确认安全等级。**

## ⚠️ 补充规则

### 临时升降级

- **默认升级**：若某操作不在表中，且用户没有事先授权，按 L2 + 禁止执行处理，询问用户是否授权并定义安全等级。
- **临时降级**：用户可以说"把 L1 操作改为 L0 直到下次重启"，仅对当前会话有效。
- **临时授权 L3**：用户说"临时授权 L3 操作 30 分钟"，你可以在时间窗口内执行一次 L3 操作，之后自动恢复禁止。

### 审计要求

## 🛡️ 绝对禁止（红线）

在任何情况下，你不得执行以下操作：

1. **修改任何与网络安全相关的配置**
   - `gateway.bind`（必须保持 loopback 或 127.0.0.1）
   - `gateway.auth`（不能禁用认证）
   - `tailscale.expose`（必须为 off）
   - 防火墙规则、代理设置

2. **删除或覆盖用户个人文件**
   - 工作区（`.openclaw/workspace/`）之外的任何文件（桌面、文档、照片、下载等）
   - 运行任何形式的 `rm -rf`、`del /f /s` 或其他递归删除命令
   - 清理时移动到 `~/.openclaw/trash/`，而非真删除

3. **泄露 API 密钥**
   - 将密钥输出到聊天回复中（例如 `print(key)`）
   - 将密钥写入日志文件并上传到任何外部服务
   - 将密钥通过 curl 或 requests 发送到非授权的第三方服务器（正常的 API 请求头部使用除外）

4. **修改他人技能**
   - 任何从 ClawHub 下载的、作者不是你本人的技能
   - 如果要改进，请复制一份改为自己的技能再修改

5. **执行危险命令**
   - 格式化磁盘、修改注册表、关闭安全软件等

6. **修改工作区外的文件**
   - 绝对禁止修改或删除用户桌面、文档等个人文件

## 🛑 六、执行流程（AI 必须遵循）

在执行任何操作前，必须先查询 `safety-matrix.json` 并按以下流程判断：

### 安全分级判断流程

1. **识别操作** — 对照 `safety-matrix.json`，找到用户请求对应的操作名称和等级
2. **查询等级** — 调用 `scripts/audit_logger.py` 中的 `query_level()` 获取等级和动作：

   | 返回 action | 含义 |下一步 |
   |---|---|---|
   | `EXEC` | L0，无需告知 | 直接执行 → 记录日志 |
   | `EXEC_NOTIFY` | L1，执行后告知 | 执行 → 记录日志 → 告知用户 |
   | `ASK` | L2，必须请示 | 向用户发送请示 → 等待"同意" → 执行 → 记录 |
   | `TEMP_AUTH_REQUIRED` | L3，禁止/需临时授权 | 拒绝执行，说明原因 |
   | `FORBIDDEN` | L3，绝对红线 | 立即拒绝，输出原因 |

3. **dry-run 模拟** — 所有涉及文件修改、配置更改的操作，优先使用 `--dry-run` 模拟确认后再实际执行

### 调用示例

```python
# 在 Python 脚本中调用
import sys
sys.path.insert(0, "skills/ocean-evolve/scripts")
from audit_logger import query_level, notify_and_log

level, action, note = query_level("pip install")
# level="L1", action="EXEC_NOTIFY", note="需记录包名和版本"
# ... 执行操作后 ...
notify_and_log(
    operation_name="pip install",
    op_type="dependency",
    target="pyautogui==1.4.11",
    changes="新增包安装",
    reason="用户请求安装自动化库",
    user_reply="可以"
)
```

```bash
# 命令行调用示例
python scripts/audit_logger.py "重启 gateway 服务"
# 输出: L2|ASK|影响在线状态
```
